## OpenAI 發布 GPT-5 預告版重點內容

OpenAI 在 2026 年 3 月底正式發布了 GPT-5 預告版，這是其繼 GPT-4 之後的重大技術突破。以下是發布會的幾個關鍵亮點：

### 主要功能亮點

1. **原生多模態能力**：GPT-5 能夠直接處理和生成圖像、音訊、影片等多模態內容

2. **超長上下文**：正式版本將支援高達 200K tokens (約 15 萬字) 的上下文處理

3. **程式生成能力大幅提升**：HumanEval 基準測試通過率達到 92.3%

4. **降低幻覺現象**：事實檢查準確度提升 40%

### 技術規格

| 項目 | GPT-4 | GPT-5 (預告版) |
|------|-------|----------------|
| 參數量 | 不公開 | 預估 2T+ |
| 上下文長度 | 32K / 128K | 200K |
| 多模態 | 分離模組 | 原生整合 |
| API 回應速度 | 基線 | 提升 2.5 倍 |

### 發布時程

- **今日**：預告版開放 API 測試
- **Q2 2026**：正式版上線
- **Q3 2026**：企業版功能開放

### 定價調整

OpenAI 也宣布了新的 API 定價策略：
- 輸入 tokens：降低 35%
- 輸出 tokens：降低 50%
- 新增多模態模組按使用量計費

### 業界反應

多位 AI 專家認為，GPT-5 的發布將進一步縮小 AI 模型與人類專家在各個領域的差距，特別是在程式開發、知識管理、創意設計等領域。
